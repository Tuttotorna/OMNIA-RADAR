# OMNIA-RADAR Scope

## Purpose

OMNIA-RADAR is a post-hoc structural detection layer.

It detects and surfaces measurable structure, residual structure, drift, instability, and persistence behavior.

It does not interpret meaning.

It does not decide truth.

## Boundary

```text
measurement != inference != decision
structure != meaning
structure != truth
```

## What is in scope

OMNIA-RADAR can scan:

- sequences
- traces
- texts
- logs
- code-like outputs
- symbolic outputs
- numeric patterns
- candidate trajectories
- residual structure near limits

It can measure:

- structural intensity
- structural persistence
- anomaly signals
- drift under perturbation
- temporal decay
- repeated pattern excess
- entropy and compression indicators

## What is out of scope

OMNIA-RADAR does not certify:

- semantic correctness
- truth
- usefulness
- coherence
- production safety
- final decisions
- model intelligence

## Correct interpretation

```text
detectable structure = structural signal
persistent structure = structural survival signal
strong structure != useful structure
strong structure != true structure
```
